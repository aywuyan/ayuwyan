第一章：认识 Markdown

Markdown 是一种轻量级标记语言，它以简洁的语法和易读易写的特点而广泛应用于写作和文档编写。使用 Markdown，我们可以快速创建格式良好的文本，包括标题、段落、列表、链接、代码块等。

例如，下面是一个使用 Markdown 的代码块示例：

def hello_world():
    print("Hello, World!")

hello_world()
